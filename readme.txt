== Description ==

AB Tasty helps you create and manage AB Tests on your website.
This plugin will automatically paste the AB Tasty code to your whole website.

== Installation ==

1. On the administration panel, go to Plugins > Add New
2. Go to the Upload section, then upload the zip file.
3. Activate the plugin.
4. From the administration panel, go to Settings > ABTasty
5. Follow the instructions to set your client ID.
